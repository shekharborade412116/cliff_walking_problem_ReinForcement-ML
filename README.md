# Cliff Walking Problem - Reinforcement Learning

This repository implements the Cliff Walking problem using two Temporal Difference (TD) Reinforcement Learning algorithms.

## Algorithms Implemented

- SARSA (State-Action-Reward-State-Action)
- Q-Learning

## SARSA Results
- Total Reward: -17
- Episode Length: 17

## Q-Learning Results
- Total Reward: -1
- Episode Length: 13

## Environment
- OpenAI Gymnasium CliffWalking-v0

## Language & Libraries
- Python
- NumPy
- Gymnasium
- Matplotlib
